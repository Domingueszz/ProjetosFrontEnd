LINGUAGEM DE PROGRAMAÇÃO | PROFº CAINÃ

Durante o semestre, exploramos conceitos fundamentais de programação utilizando Java.

Java é uma linguagem versátil e amplamente utilizada na indústria de desenvolvimento de software, conhecida por sua portabilidade e segurança.

Aprendemos sobre os princípios da programação orientada a objetos, como encapsulamento, herança, polimorfismo e abstração, essenciais para criar código modular e reutilizável.

Utilizamos a plataforma Intellij IDEA para desenvolvimento, um ambiente integrado robusto que oferece ferramentas avançadas para facilitar o desenvolvimento Java.

Além disso, exploramos JavaFX para a criação de interfaces gráficas de usuário (GUIs), ]permitindo-nos desenvolver aplicativos desktop interativos e visualmente "atraentes". A integração com o GitHub foi fundamental para gerenciar e colaborar em projetos de forma eficiente, utilizando controle de versão para acompanhar mudanças no código e facilitar o trabalho em equipe.

PROJETOS

Projeto Screensaver: ━━━━━━━━ ▣

O projeto Screensaver, desenvolvido em JavaFX no IntelliJ, exemplifica o uso de classes e métodos para criar um protetor de tela dinâmico e visualmente interessante. O objetivo do projeto é exibir uma tela cheia de formas geométricas - quadrados, círculos, elipses e retângulos – que se movem e ricocheteiam nas bordas da tela.

Para implementar este projeto, criamos várias classes representando cada forma geométrica. Cada classe contém métodos para desenhar a forma, atualizar sua posição e detectar colisões com as bordas da tela.
A lógica de movimento e ricocheteamento é gerida por um método que verifica continuamente a posição das formas e ajusta suas trajetórias quando atingem as bordas da janela.

No IntelliJ, organizamos o projeto utilizando boas práticas de programação orientada a objetos, encapsulando as propriedades e comportamentos das formas em suas respectivas classes e utilizando métodos para modularizar a lógica de movimento e colisão.

Projeto RPG: ————————————

O projeto RPG, desenvolvido em Java no IntelliJ, consiste em uma simulação simples de um jogo de luta. O jogo permite que jogadores definem os personagens e lutem entre si.

A classe principal, Personagem, representa um lutador no jogo com atributos como nome, vida, ataque e defesa. Além disso, a classe Personagem inclui um método ataque, que permite aos personagens atacar seus adversários. O método ataque calcula o dano com base nos atributos de ataque do atacante e defesa do defensor, reduzindo a vida do adversário adequadamente.

Esse projeto demonstra a aplicação de conceitos fundamentais de programação orientada a objetos, como encapsulamento e métodos, proporcionando uma base sólida para o desenvolvimento de jogos mais complexos no futuro.

Projeto Game Java: ━━━━━━━━━ ▣

O projeto será realizado em squads e consiste na elaboração de um game em Java, utilizando JavaFX no IntelliJ. A equipe irá colaborar para desenvolver um jogo interativo, aproveitando as capacidades gráficas e de interface do JavaFX.

O objetivo do projeto é criar um jogo com uma jogabilidade básica, que pode incluir elementos como personagens controláveis, obstáculos, pontuações e níveis.

Cada squad será responsável por diferentes aspectos do desenvolvimento, como design de interface, lógica de jogo, e integração de funcionalidades.

Utilizando Java e JavaFX, o projeto permitirá explorar conceitos avançados de programação orientada a objetos, desenvolvimento de interfaces gráficas, e trabalho em equipe com ferramentas de controle de versão como Git. O resultado será um jogo funcional que demonstra habilidades técnicas e de colaboração em equipe.

CONTEÚDOS

Listas:



O conjunto de atividades consiste em seis listas de exercícios diferentes, abordando temas essenciais da programação com Java. Cada lista foca em um aspecto específico da linguagem, proporcionando uma base sólida e progressiva para o aprendizado.

✔︎ Variáveis: Esta lista introduz os conceitos básicos de variáveis, tipos de dados, e operações fundamentais. Os exercícios ajudam a entender como declarar, inicializar e manipular variáveis em Java.

✔︎ Estruturas de Decisão: Focada em estruturas condicionais, esta lista ensina a usar if, else if, else, e switch para tomar decisões no código com base em diferentes condições.

✔︎ Estruturas de Repetição: Aqui, os exercícios abordam os laços for, while, e do-while, permitindo a repetição de blocos de código até que certas condições sejam satisfeitas, essenciais para tarefas iterativas.

✔︎ Vetores: Esta lista explora arrays (vetores), ensinando como declarar, inicializar, e manipular sequências de elementos, além de abordar iteração e operações comuns sobre arrays.

✔︎ Funções: Focada em métodos (funções), esta lista ensina a criar e usar métodos para organizar e reutilizar código, passando parâmetros e retornando valores.

✔︎ Orientação a Objetos: A última lista introduz os princípios da programação orientada a objetos, incluindo classes, objetos, herança, polimorfismo, encapsulamento e abstração. Os exercícios ajudam a entender como modelar problemas reais utilizando esses conceitos.
