Neste primeiro projeto de Desenvolvimento Web, desenvolvemos uma calculadora de Índice de Massa Corporal (IMC) utilizando HTML, CSS e JavaScript.

O objetivo foi aplicar conceitos básicos de criação de páginas web e interatividade. Estruturamos a página com campos de entrada para peso e altura, um botão para calcular e uma área para exibir o resultado.

A calculadora permite ao usuário inserir seu peso (em kg) e altura (em metros), realizar o cálculo do IMC ao clicar em um botão, e visualizar o resultado junto com uma breve interpretação (Obessidade, abaixo do peso e peso ideal).

Neste projeto de Desenvolvimento Web, criamos uma calculadora que suporta operações matemáticas utilizando três notações distintas: infix, prefix (pré-fixa) e postfix (pós-fixa).

Utilizamos HTML para estruturar a interface, CSS para estilizar os elementos visuais e JavaScript para implementar a lógica de cada notação.

Estrutura:

❱ Infixa: A notação padrão em que operadores estão entre os operandos (por exemplo, 2 + 3 * 4).

❱ Prefixa: Também conhecida como notação pré-fixa, em que o operador precede os operandos (por exemplo, + 2 * 3 4).

❱ Pós-fixa: Também conhecida como notação pós-fixa ou polonesa reversa, onde o operador segue os operandos (por exemplo, 2 3 4 * +).

CONTEÚDOS

CSS

CSS, ou Cascading Style Sheets, é a linguagem utilizada para estilizar e controlar a apresentação visual de páginas web. Ao contrário do HTML, o CSS foca em como esse conteúdo é apresentado ao usuário.

Utilizando regras de estilo, o CSS permite definir cores, fontes, espaçamentos, layouts e outros aspectos visuais de elementos HTML. Isso é feito através de seletores que identificam quais elementos da página serão estilizados e propriedades que especificam como eles devem ser apresentados.

A principal vantagem do CSS é sua capacidade de separar o conteúdo (HTML) da apresentação visual, o que proporciona maior flexibilidade e facilita a manutenção do código. Isso significa que você pode alterar a aparência de uma página inteira modificando apenas um arquivo CSS, sem precisar alterar o HTML.

HTML

HTML, ou HyperText Markup Language, é a linguagem fundamental utilizada para criar páginas na web. Ela funciona através de uma estrutura de marcação, onde cada elemento é envolvido por tags que definem sua função e apresentação. Essas tags permitem organizar o conteúdo de uma página web de forma hierárquica e semântica.

Além de estruturar o conteúdo básico, como textos, imagens e links, o HTML moderno oferece elementos semânticos que ajudam na acessibilidade e indexação por mecanismos de busca.

HTML trabalha em conjunto com CSS (Cascading Style Sheets) para estilizar o conteúdo e JavaScript para adicionar interatividade e comportamento dinâmico às páginas. Essa combinação de tecnologias permite criar desde páginas estáticas simples até aplicações web complexas e responsivas.

JS

JavaScript, conhecido como JS, é uma linguagem de programação essencial para o desenvolvimento web.
Ao contrário do HTML e do CSS, o JavaScript adiciona interatividade e funcionalidade dinâmica às aplicações web.

Ele permite criar botões que respondem a cliques, validar formulários antes de enviá-los, e modificar o conteúdo de uma página em tempo real.

Com o JavaScript, é possível também fazer requisições assíncronas para carregar dados sem recarregar a página inteira.
Além de ser usado no front-end das aplicações web, o JavaScript também é utilizado no back-end com plataformas como Node.js.
Possibilitando o desenvolvimento de aplicações full-stack usando apenas uma linguagem de programação. JS desempenha um papel crucial na criação de experiências web dinâmica.
